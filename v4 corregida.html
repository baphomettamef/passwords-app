Crea una aplicación web interactiva y atractiva para que niños de 7 a 9 años practiquen las tablas de multiplicar. La aplicación debe ser colorida, amigable e intuitiva. Características esenciales: 1.Ejercicios dinámicos: Genera multiplicaciones aleatorias del 1 al 9. 2.Interacción: Permite al usuario ingresar su respuesta en un campo de texto. 3.Validación y retroalimentación: -Comprueba si la respuesta es correcta o incorrecta. -Muestra un mensaje motivador en cada caso. 4.Sistema de puntuación: Lleva la cuenta de aciertos y errores. Muestra la puntuación en tiempo real. 5.Diseño pensado para niños: Usa colores llamativos, fuentes grandes y botones accesibles. Incluye animaciones suaves para hacer la experiencia más atractiva. 6.Diseño responsivo: La aplicación debe ser completamente adaptable a móviles, tabletas y computadoras utilizando Tailwind CSS. 7.Código estructurado en un solo archivo: Incluir HTML, CSS y JavaScript en un solo archivo .html. 8.Fácil ejecución: El archivo debe ser ejecutable con un doble clic (sin necesidad de servidores adicionales). Debe incluir instrucciones claras en un comentario dentro del código sobre cómo usar la aplicación. 9.Extras opcionales (si es posible): Agregar sonidos o efectos visuales para mejorar la experiencia. Implementar un botón para reiniciar la partida. Genera el código completo asegurando que sea funcional, atractivo y fácil de entender.
